Hi, my name is Yash Agrawal and I'm an ECE major at the University of Illinois at Urbana-Champaign. I love Frontend, Running and watching movies in languages I don't understand.

To me, frontend is an intersection of my two interests- Computer Science and Design and it encompasses the best of both worlds. JavaScript is one of the best things ever made and I love reading and writing about it.

Frontend development has gotten a lot more complicated today with a bunch of libraries popping up every second. (A few more were published by the time you finished reading that sentence). While they are mostly helpful, and powerful, it is always good to know the basics because all these libraries are basically abstracted fundamentals.

Keeping that in mind, I made this website using pure CSS, HTML and JS without using any libraries like Bootstrap for example. It is responsive and has cross browser support.

I've worked on personal projects that are frontend focused.
